Introduction to PHP




  • PHP stands for PHP: Hypertext Preprocessor.PHP has become a widely used scripting language among web developers.
  • One can use PHP by first installing a PHP development enviorenment. I recomend downloading XAMPP, you can download XAMPP by clicking the link below.
  • Click here to download XAMPP (PHP development enviorenment).
  • After downloading XAMPP, start the application and turn the Apache server on. This allows us to run PHP code on our local host.
  • Now, you want to create a file such as welcome.php within this directory "C:\xampp\htdocs\".
  • Now you can insert any php code into your file, make sure you open and close your code witin a php tag.
  • Lastly, if you named your file welcome.php, type/copy the link below into your browser.
  • http://localhost/welcome.php
  • If you want to declare variables in PHP, you can do so by using the $ sign. Below is an example of me declaring a variable.
  • $x = 9
